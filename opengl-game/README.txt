q character exits game.
r character restarts game.
left mouse click pauses game, if left mouse is clicked the second time game resumes.
right mouse click leads to debugging mode if is in a running state. (to use debugger mode, the game is not to be over yet or paused already)
continuosly clicking right mouse leads to increase current state by 1 while debugging thus the vehicles will be updated also in their next state.
if the agent comes back to the lane which it had started at the first place, the difficulty level of the game increases by 1. (faster vehicles)
the amount of the coins that are taken are also displayed along the score.
if the direction of the agent is not the same as the entered arrow key then the game is over.
